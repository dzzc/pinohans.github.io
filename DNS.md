# 1. DNS记录类型

| 记录类型  | 说明                                           |
| --------- | ---------------------------------------------- |
| A记录     | IP地址记录，记录一个域名对应的IP地址           |
| AAAA记录  | IPv6地址记录，记录一个域名对应的IPv6地址       |
| CNAME记录 | 别名记录，记录一个主机的别名                   |
| MX记录    | 电子邮件交换记录，记录一个邮件域名对应的IP地址 |
| NS记录    | 域名服务器记录 ,记录该域名由哪台域名服务器解析 |
| PTR记录   | 反向记录，也即从IP地址到域名的一条记录         |
| TXT记录   | 记录域名的相关文本信息                         |
| SOA记录   | start of anthorization 开始授权                |
| SRV记录   | 记录提供服务的信息。                           |
